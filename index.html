What is 'MORA GPA' ?

'MORA GPA' is a simple web based application which is capable of calculating the grade point averages(GPA) of Engineering Faculty Of University Of Moratuwa. This website based on the module data related to academic year
2016/2017.

How to use ?

First select your semester then select your deparrtmet(If you have). You can see a table which including details of the modules. "Module Code","Module Name","Credit","Type","Grade" these are the table data sections. Here you have to select module type. there are four types "GPA", "NGPA","Not Selected","No Exam". then you have to select you grades for each module. If you already have a c CGPA please add it on to "CGPA" field and hit the "Calculate my new GPA" button.then web display the your GPAs. If you need to calculate next semester GPA git "Goto Next Semester" button.
